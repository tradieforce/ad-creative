# /assets/generated-ads — final ChatGPT outputs

Empty until ChatGPT Image 2 is connected and ad generation runs.

## Folder convention

```
generated-ads/{client-slug}/
├── ad_001.png         ← ad-id matches data/ads.json
├── ad_002.png
└── ...
```

## What's saved here

The PNG files generated by ChatGPT Image 2. Typically 1080×1080 (or 1080×1350 for A9 holiday portrait variant per HR10).

## Storage scaling

In production, every client × every archetype × every generation = many images per client per pack. This folder grows fast.

The tech guy's job is to move this to S3/R2 and not store binaries in the local filesystem long-term.
